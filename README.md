# Audio Splitter Pro

Audio Splitter Pro es una aplicación web que permite dividir archivos de audio largos en múltiples segmentos de forma automática.  
El sistema procesa el archivo y genera versiones optimizadas en formato **MP3**, reduciendo el tamaño y facilitando su uso, almacenamiento o distribución.

## Demo
https://audio-splitter-pro-production.up.railway.app/

## Características

- División automática de archivos de audio largos
- Conversión a formato **MP3** para reducir el tamaño del archivo
- Procesamiento rápido de audio
- Interfaz web simple y fácil de usar
- Ideal para trabajar con grabaciones extensas

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Procesamiento de audio

## Cómo usar la aplicación

1. Cargar un archivo de audio.
2. El sistema procesa el archivo automáticamente.
3. El audio se divide en múltiples segmentos.
4. Los segmentos se generan en formato **MP3**.
5. Descargar los archivos generados.

## Objetivo del proyecto

Este proyecto fue desarrollado como una herramienta para automatizar el proceso de dividir archivos de audio largos en partes más pequeñas, facilitando su uso en diferentes aplicaciones como edición de audio, publicación de contenido o procesamiento posterior.
